# De patria a patria (web_project_homeland)

Sitio web responsivo de una sola página para **Tripleten Art Gallery**. Presenta un encabezado con el logotipo, una sección principal (lead) con título, subtítulo e imagen, un bloque de introducción con texto y cita, y un pie de página.

## Funcionalidad
- Diseño adaptativo para 1280px, 768px y 320px.
- Tipografía **Inter** (variable font) cargada localmente.
- Estructura de estilos con metodología **BEM** (estructura plana en `/blocks`).

## Tecnologías y técnicas
- HTML5 semántico (`header`, `main`, `section`, `footer`).
- CSS: Flexbox, `@media` queries, `calc()` para anchos adaptativos.
- Normalización de estilos con `normalize.css`.
